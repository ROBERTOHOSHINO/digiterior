UI-FLORIST kintone移行 —「ソロ＋Claude」を顧客へ手渡す
SQL Server + C# の旧システムをkintoneへ移行。最大の難所「帳票印刷」は、担当者 × Claude で旧C#プログラムを一画面ずつHTML化。設計の核心は性質による切り分け——帳票（完成品）はPDFで固定し、統計（素材）はExcelで素のまま出力。そのExcelをユーザー自身がClaudeに相談しながら自由に加工する。ダンクソフトが "ソロ＋Claude" に畳んだ構造を、顧客の "自分＋Claude" へ相似形で手渡した初の実装例。
